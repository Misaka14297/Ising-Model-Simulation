# 2D Ising Model Simulation & Wolff Algorithm Optimization

本项目源自我的本科毕业论文，主要研究二维 Ising 模型的相变行为及算法效率优化。

## 核心功能
* **Metropolis 算法实现**：经典的局部自旋翻转模拟。
* **Wolff 集群算法实现**：基于队列管理的自旋簇翻转，显著缓解临界减速效应。
* **有限尺寸标度分析 (FSS)**：计算不同尺寸 (L=16, 32, 64) 下的比热、磁化率及临界温度。

## 关键结果
* Wolff 算法在 $T_c$ 附近将能量标准差降低了约 **60%**。
* 模拟得到的临界温度 $T_c$ 与理论值（2.269）偏差小于 **0.9%**。
